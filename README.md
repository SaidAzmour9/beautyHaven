
# Cosmetic Management and AI

Cosmetic Management and AI is a web application designed to help users find the best cosmetic products based on their skin type. The project leverages machine learning to provide personalized recommendations and offers functionalities to manage user orders.
## Features

- User Authentication: Signup and login functionality using SQLite3.
- Product Recommendation System: Recommends products based on skin type using an SVM model with TF-IDF vectorization.
- Order Management: Allows users to place and view their orders.
- Product Database Management: Manage cosmetic product information including brand, price, and others.


# Technologies Used
### Backend
- Flask, SQLite3
### Machine Learning
- SVM, TF-IDF
### Frontend
- HTML, CSS, JAVASCRIPT, BOOTSTRAP

## Run Locally

Clone the project

```bash
  git clone https://github.com/SaidAzmour9/beautyHaven
```

```bash
  cd beatyHaven
```
Create and activate a virtual environment(Windows):

```bash
  python3 -m venv venv
  venv\Scripts\activate
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python run.py
```
